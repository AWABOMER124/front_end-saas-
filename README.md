# Ideal Agent Dashboard (Arabic RTL)

واجهة SaaS جاهزة للإنتاج لمنصة Ideal Agent من Perfect Team، مبنية بـ Next.js (App Router) و Tailwind.

## البدء محلياً
1. ثبّت الاعتماديات:
   ```bash
   npm install
   ```
2. أنشئ ملف بيئة:
   ```bash
   cp .env.example .env
   ```
3. شغّل التطبيق:
   ```bash
   npm run dev
   ```
4. افتح المتصفح على `http://localhost:3000`.

> بيانات الدخول والحالة محفوظة محلياً في LocalStorage ضمن وضع العرض التجريبي.

## التقنيات
- Next.js + TypeScript (App Router)
- Tailwind CSS + مكونات مبسطة مستوحاة من shadcn/ui
- lucide-react للأيقونات
- Recharts للرسوم البيانية
- دعم RTL كامل، نسخ عربية واضحة

## معاينة الثيم
- صفحة مخصصة لاختبار الأزرار والبطاقات والجداول والرسوم: توجه إلى `/theme-preview` بعد تشغيل الخادم.
